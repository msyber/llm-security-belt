Playbook (3 steps)
  1. Gateway: sanitize I/O, rate-limits, URL allowlist, logs.
  2. Policies: deny-by-default tools/data, no secrets in prompts, outputs treated as untrusted (JSON schema).
  3. Evals in CI: injection + jailbreak tests; fail the build on threshold.
